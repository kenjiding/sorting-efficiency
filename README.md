# 生产力跟踪管理系统 🏢

上传S3命令: AWS_PROFILE=aws_872570730 aws s3 cp ./dist s3://sorting-efficiency --recursive

一个专业的企业级生产力跟踪管理系统，采用现代化的Web技术栈构建，支持数据录入、分析报表和实时监控。

- **左侧导航栏**：可收缩的侧边栏，包含菜单项和实时统计

## 🎯 使用指南

### 快速开始

1. **安装依赖**
   ```bash
   npm install
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   ```

3. **访问应用**
   ```
   http://localhost:5173
   ```

### 数据录入

#### 手动录入
1. 点击左侧菜单"数据录入"
2. 选择"手动录入"标签
3. 填写基本信息：日期、姓名
4. 填写工作详情：粗拣和细拣的时间及数量
5. 点击"添加记录"保存

#### Excel导入
1. 选择"Excel导入"标签
2. 下载Excel模板（可选）
3. 准备符合格式的Excel文件
4. 上传文件并预览数据
5. 确认导入

### 数据分析

#### 查看报表
1. 点击左侧菜单"数据分析"
2. 使用筛选器选择时间段和工人
3. 查看统计卡片和图表
4. 导出Excel或CSV报告

#### 编辑数据
1. 在数据表格中点击"编辑"按钮
2. 在弹出的模态框中修改数据
3. 点击"保存更改"确认

#### 删除数据
1. 在数据表格中点击"删除"按钮
2. 确认删除操作

## 📊 数据结构

### 记录字段
```javascript
{
  id: Number,              // 自动生成ID
  date: String,            // 日期 (YYYY-MM-DD)
  name: String,            // 工人姓名
  coarseStartTime: String, // 粗拣开始时间 (HH:mm)
  coarseEndTime: String,   // 粗拣结束时间 (HH:mm)
  fineStartTime: String,   // 细拣开始时间 (HH:mm)
  fineEndTime: String,     // 细拣结束时间 (HH:mm)
  coarseCount: Number,     // 粗拣数量
  fineCount: Number,       // 细拣数量
  coarseEfficiency: String,    // 粗拣效率 (自动计算)
  fineEfficiency: String,      // 细拣效率 (自动计算)
  totalWorkingHours: String    // 总工作时长 (自动计算)
}
```

### Excel导入格式
支持以下列名（中英文均可）：
- 日期/Date
- 姓名/Name
- 粗拣开始时间/Coarse Start Time
- 粗拣结束时间/Coarse End Time
- 细拣开始时间/Fine Start Time
- 细拣结束时间/Fine End Time
- 粗拣数量/Coarse Count
- 细拣数量/Fine Count
